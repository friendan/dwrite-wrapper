DWrite Wrapper
==============

DWrite.dll TextRenderingParams and TextAntialiasMode wrapper.

Quick Start
-----------
*Run as administrator*

Install:

    > ie-install.bat

Uninstall:

    > ie-uninstall.bat

Build
-----
32bit:

    > nmake -f Makefile clean install

64bit:

    > nmake -f Makefile.x64 clean install
